## 作用域区分

| 维度 | references/standards/ | skills/*/references/ |
|------|---------------------|---------------------|
| **作用域** | 全局，所有技能共同遵守 | 技能特定，仅该技能使用 |
| **内容类型** | 复杂操作规范、环境配置 | 参考文档、模板、示例 |

## 标准间引用原则（SSOT）

- 标准文档之间可以存在**操作依赖**（如 P4 依赖 G10 的 CLI 启动流程）
- 每个标准的**概念定义、参数合同、完成条件**仍以各自文件为 SSOT
- 引用其他标准时，仅说明"依赖关系"，不复制其内容
